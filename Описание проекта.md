Проект курсор внутри докер контейнера.

**Подробная документация:** папка [Документация](Документация/README.md).

Свойства
1) В качестве образа использовать debian bookworm slim
2) В качестве визуально среды x11 
3) Для подключения использовать xvnc server
4) Должен быть установлен cursor
5) cursor должен обновляться 
6) Оболочка bash
7) Должен быть установлен google chrome
Образ должен быть описан в докер файле
Подключение по vnc без пароля
Для создания контейнера необходимо описание в виде
docker compose файла 
подключение только на localhost:9999
Монитировать папки в образ использовать bind
Профиль google chrome и cursor должен быть примонтирован в контейнер
При пересоздании контейнера(docker compose down && docker compose up -d)
Google chrome должен загружаться с примонтированым профилем
cursor также должен загружаться с примонтированым профилем
